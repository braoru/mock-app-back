Mock App Back-end
==================

## Install
Go installer can be found [here](https://golang.org/doc/install).

## Build
Gorilla Mux is needed and can be installed with the following command:
```
go get github.com/gorilla/mux
```
To build the mock:
```
cd cmd
go build
```

## Run
```
cd cmd
./cmd
```

## Run tests
```
cd pkg
go test
```
